English

Contact

Support is handled through GitHub Issues: https://github.com/Facta-Leopard/BunMixer/issues.

Before Requesting Support

  • System Audio Recording permission is required only after you explicitly enable mixing or start the Safari test flow.
  • Audio is processed locally and is not recorded, saved, uploaded, analyzed, transcribed, or transmitted.
  • Some apps, helper processes, output devices, protected content, or system permission states may prevent per-app volume control.
  • Use Pause Mixing, Reset All, or disable mixing in settings to stop active processing.
  • Revoke System Audio Recording permission in System Settings if you no longer want BunMixer to control supported app playback audio.

Antes de solicitar soporte

Deutsch

Kontakt

Support erfolgt über GitHub Issues: https://github.com/Facta-Leopard/BunMixer/issues.

Vor einer Support-Anfrage

  • Die Berechtigung für Systemaudioaufnahme ist erst erforderlich, nachdem du das Mischen ausdrücklich aktivierst oder den Safari-Testablauf startest.
  • Audio wird lokal verarbeitet und nicht aufgenommen, gespeichert, hochgeladen, analysiert, transkribiert oder übertragen.
  • Einige Apps, Hilfsprozesse, Ausgabegeräte, geschützte Inhalte oder Systemberechtigungszustände können die App-Lautstärkesteuerung verhindern.
  • Verwende Mischen pausieren, Alle zurücksetzen oder deaktiviere das Mischen in den Einstellungen, um aktive Verarbeitung zu stoppen.
  • Widerrufe die Berechtigung für Systemaudioaufnahme in den Systemeinstellungen, wenn BunMixer keine Wiedergabe von unterstützten Apps mehr steuern soll.
